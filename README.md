# Mid-VAP Assignment

## Course registration system

### Run Instructions
```
npm start
```

### Technology used
* HTML
* Node JS
* Pug
* Bootstrap

### Functionalities
The system is used as an interaction portal for the addition and enrollment in courses for students and professors

* Student
    * Login
    * View Already enrolled Courses
    * Enroll in a New Course
* Professor
    * Login
    * View Courses taught
    * View Students
    * Add a new course
